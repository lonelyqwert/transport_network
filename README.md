# Транспорт: кратчайший маршрут

## Запуск
1. Установить зависимости: `pip install -r requirements.txt`
2. Запустить: `python main.py`

## Промт для генерации данных
Сгенерируй два связанных CSV-файла на тему "Транспортная сеть города".

Файл 1: stops.csv
Поля:
- stop_id (int, автоинкремент с 1)
- name (str, название остановки на русском)
- district (str, один из: Center, North, South, East, West)
- lat (float, широта в диапазоне 57.60 – 57.65)
- lon (float, долгота в диапазоне 39.85 – 39.92)

Количество записей: 50
Ограничения:
- Добавить 4 пропущенных значения: в 2 строках пропустить name, в 2 строках пропустить district.
- Добавить 1 дубликат (полностью одинаковую строку) для какой-нибудь остановки.
- Разнообразие названий и районов.

Файл 2: links.csv
Поля:
- link_id (int, автоинкремент с 1)
- from_stop_id (int, ссылается на stop_id)
- to_stop_id (int, ссылается на stop_id)
- travel_minutes (int, от 2 до 12)
- line_number (int, от 1 до 20)

Количество записей: 150
Связь с первым файлом: from_stop_id и to_stop_id — FOREIGN KEY на stops.stop_id.
Ограничения:
- Граф ориентированный: для каждой связи создай обратную.
- Добавить 3 пропущенных значения: в 3 строках пропустить travel_minutes.
- Добавить 1 дубликат (полностью одинаковую строку) для какой-нибудь связи.
- Все остановки должны быть связаны в единую сеть.

Верни два отдельных CSV в виде раздельных блоков кода, без пояснений между ними. Заголовок — первая строка каждого файла.
